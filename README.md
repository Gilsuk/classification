# text-classification

엑셀 파일을 읽어들여 문장 데이터를 얻는다.

형태소 분석기를 통하여 문장을 토큰화하고 정수화 임베딩 전처리를 한다.

RNN, 나이브베이즈 알고리즘으로 기계학습을 각각 진행하여 모델을 생성한다.

생성된 모델을 통하여 문장을 분류한다.

## 사용도구

Jupyterlab, Konlpy, keras, gensim
